# OctoAcme — Project Status Update Template

Use this template for regular project status updates shared with stakeholders. The **Project Manager (PM)** owns the status update and distributes it on the agreed cadence (typically weekly or milestone-based). It serves as the single source of truth for project health and is referenced by [`octoacme-risks-and-communication.md`](octoacme-risks-and-communication.md).
